# swift-vip-template
[Generamba](https://github.com/rambler-ios/Generamba) templetes for VIP (View->Interactor->Presenter) modules

### Usage

1. Add to your `Rambafile`:
  ```yaml
  catalogs:
  - 'https://github.com/somedev/swift-vip-template'
  
  templates:
  - {name: swift_vip}
  ```
2. Run `generamba template install`
